Fuentes

Elegir una fuente apropiada tiene un fuerte impacto en como los lectores, experimentan un sitio web.

La fuente correcta puede crear una reconocible entidad para tú marca o página web.

Usar una fuente fácil de leer es muy importante.Ya que añade valor a tu texto. También es importante elegir un correcto color y tamaño.

Familia de fuentes genéricas

En CSS hay 5 familias genéricas:

  1. Fuentes con serif tienen un pequeño trazo en las puntas de cada letra.
  2. Fuentes Sans-serif.No tienen los trazos de las anteriores y tienen un aspecto minimalista y moderno.
  3. Monospace. Todas las letras tienen la misma anchura. Crean un aspecto mecánico. La fuente que vemos en nustro código es de este tipo.
  4. Cursivas imitan la caligrafría humana.
  5. Fantasy.Tienen un aspecto decorativo juguetón.

    Tamaño de fuente

    La propiedad Font-size especifica el tamaño de un texto.

    Ser capaz de modificar los tamaños de los diferentes textos de una web es muy importante. Sin embargo, no uses párrafos con tamaños mayores para que parezcan titúlos y viceversa. Usa las correspodientes etiquetas h1-h6 para titúlos o P para párrafos.

    Los valores de tamaño puede ser Absolutos o relativos:

    Valor absoluto:

    • Establece el texto a un tamaño específico.
    • No permite que el usuario cambie el tamaño del texto desde su navegado (malo por razones de accesibilidad)
    • Un valor absoluto es útil cuando se conoce el tamaño físico del dispositivo con el que se visualizará nuestra página.

    Valor relativo:

    • Establece el tamaño relativo a los elementosde alrededor.
    • Permite al usuario cambiar el tamaño del texto en el navegador

    Fuentes responsive

    Podemos usar la unidad REM para conseguir directamente el efecto de de usar la combinación de Em y body *font-size:100%*

    Otra medida responsive es la unidad VW,que significa "viewport widht" o ancho de la pantalla.

    Texto reponsive

    Cambia el tamaño de la ventana para ver el efecto

    Usa la unidad "VM" para darle tamaño al texto. por ejemplo, 10W pondrá el texto al 10% del tamaño del ancho de la pantalla.

    viewport es el tamaño de la ventana del navegador. 1 vw equivale al 1% del ancho de la ventana. Si la ventana mide 50cm de ancho, 1w, será 0.5cm de ancho.
