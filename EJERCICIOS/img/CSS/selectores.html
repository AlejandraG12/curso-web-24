Selectores en CSS

Los selectores se usan para "encontrar" los elementos HTML a los que queremos otorgar de estilo

Tienen 5 categorias

  • Selectores simples:nombre , clase, id.
  • Selectores combinadores:Seleccionan elementos basandose en una relación especifica entre ellos.
  • Selectores de PSeudo -clase seleccionan elementos basandose en un estadoespecifico.
  • Selectores de Pseudo-elementos seleccionan y estiliszan ciertas partes de un elemento.
  • Selectores de atributosseleccionan elementos básandose en un atributo o en el valor de un atributo.

Selectore id

Selecciona un único elemento básandose en el atributo id.

El atributo id es único por cada elemento. Si en el documento hay dos elementos con la mima id, se aplicará al útlimo(orden de lectura).

Selector de clase

Selector Universal

Selecciona TODOS los elementos del documento. El selector es un asterisco *.

Selectores Combinadores

Un combinador es un selector que explica la relación entre dos elementos.

Hay cuatro tipos distintos de combinadores:

  • Selector descendiente (espacio)
  • Selector child o hijo (>)
  • Selector adjacente o hermano (+)
  • Selector de hermano general (~)
