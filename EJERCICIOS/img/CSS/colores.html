Colores en CSS

Hay tres manera de definir colores en CSS: por nombre,RBG, y hexadecimal.

Por nombre:

CSS tiene una lista de colores predefinidos que podemos utilizar simplemente al nombralos en el código, ejemplo: dodgerblue

por RGB

RGB significa red, Green, blue. Cada valor va de 0 a 255 y los colores se cpnsiguen combinando al rojo, verde, azul ejemplo ROSA

Hexadecimal

Hexadecimal se representa con una (#) seguida de 6 números (#RRGGBB), cada par de números ejemplo: VERDE
