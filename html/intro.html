Introducción a HTML

Qué es HTML

HTML significa Hyper Text Language.En castellano, lenguaje de marca de hipertexto.

Html describe la estructura de una pàgina web. Consiste en una serie de elementos que indica al navegador cómo mostrar la página.

los elementos marcan los trozos de contenido y lo describen como "Esto es un párrafo", "esto es un título", "esto es una imagen/link/botón"...

¿Qué es un elemento?

Esctructura de un elemento

contenido

Un elemento es todo aquello desde la etiqueta de apertura hasta la etiqueta de cierre.Casi todos los elementos están formados por etiquetas de apertura y cierre salvo excepciones.

EJEMPLO:
La etiqueta"br" es una etiqueta vacía, es decir,no tiene contenido ni etiqueta de cierre.Sirve para indicar un salto de línea.
