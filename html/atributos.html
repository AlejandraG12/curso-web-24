Todos los elementos pueden contener atributos proveen información adiccional sobre los elementos.

Los atributos se especifican en la etiqueta apertura. se componen de dos partes, nombre y valor.

src

La etiqueta o elemento (img) se usa para insertar una imagen en una página HTML. El atributo src indica el camino o dirección de la imagen a mostrar.

Widht y height

Los atributos Widht, ancho y heigh, altura, definene el tamaño que queremos otorgar a la imagen de nuestra pagína.

alt

El atributo alt define el texto alternativo para una iamge. Este texto te mostrará si por alguna razón, la imagen no puede cargarse.
