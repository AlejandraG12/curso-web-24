Seleccionar elementos Html

Frecuentemente, vamos a querer manipular elementos con Javascript. Para ello deberás, antes que nada, encontrarlos e identificarlos.

Hay varias maneras esto:

  • Encontrar elementos HTML por id.
  • Encontrar elementos HTML por clase.
  • Encontrar elementos HTML por nombre de etiquetas.
  • Encontrat elementos HTML por selectores de CSS.
